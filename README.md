# CityRide Prototype

**Real-Time Public Transport Tracking App**

---

## Overview

CityRide is a prototype mobile and web application designed for real-time tracking of public transport vehicles in small cities. It aims to reduce commuter wait times, improve transport reliability, and help municipal authorities manage fleets efficiently.

---

## Features

- Live GPS tracking of buses using driver smartphones or GPS devices  
- Rider app displays live location, ETA, and route alerts  
- Driver app for trip status, telemetry, and live location updates  
- Admin dashboard for monitoring routes, vehicle status, and alerts
